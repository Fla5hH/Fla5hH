### 1、循环语句
    
    
    while 判断条件(condition)：
        执行语句(statements)……
    
    
    for <variable变量> in <sequence顺序>:
        <statements表达式>
    else:
        <statements表达式>
    
    
    for i in range(5):
    ...     print(i)
    ...
    输出：
    0
    1
    2
    3
    4
    
    
    for i in range(0, 10, 3) :
        print(i)
    从0-9 以步长为三打印   
    0
    3
    6
    9
    >>>

注意循环中的引号

### 3、推导式

推导式就是把一个for循环放在一行里搞定

从原先所在的“可迭代对象”（例如字典，集合）中生成一个新的数据序列
    
    
    [表达式 for 变量 in 列表] 
    [out_exp_res for out_exp in input_list]
    或者 
    [表达式 for 变量 in 列表 if 条件]
    [out_exp_res for out_exp in input_list if condition]
    
    
    >>> names = ['Bob','Tom','alice','Jerry','Wendy','Smith']
    >>> new_names = [name.upper()for name in names if len(name)>3]
    >>> print(new_names)
    ['ALICE', 'JERRY', 'WENDY', 'SMITH']

### 4、异常处理
    
    
    while True:
        try:
            x = int(input("请输入一个数字: "))
            break
        except ValueError:
            print("您输入的不是数字，请再次尝试输入！")

如果输入的不是整数，则会跳到except语句处执行
    
    
    except (RuntimeError, TypeError, NameError):
        pass

也可以多个错误采用一个方法处理
    
    
    try:
        runoob()
    except AssertionError as error:
        print(error)
    else:
        try:
            with open('file.log') as file:
                read_data = file.read()
        except FileNotFoundError as fnf_error:
            print(fnf_error)
    finally:
        print('这句话，无论异常是否发生都会执行。')

finally无论是否有错误都会执行

### 5、字符串
    
    
           char = 'f l a s h b a n g'
    字符串对应索引  0 1 2 3 4 5 6 6 7
    
    
    str[0] = f
    str[5] = b
    : 冒号表示从当前索引前或后截断（左闭右开）
    str[:3] = 'fla' 表示截断到索引3之前
    str[1:7] = 'lashban' 表示从索引1开始 截断到7之前

字符串格式化，语法与C语言一样
    
    
    print ("我叫 %s 今年 %d 岁!" % ('小明', 10))

format字符串格式化函数
    
    
    >>>"{} {}".format("hello", "world")    # 不设置指定位置，按默认顺序
    'hello world'
     
    >>> "{0} {1}".format("hello", "world")  # 设置指定位置
    'hello world'
     
    >>> "{1} {0} {1}".format("hello", "world")  # 设置指定位置
    'world hello world'

可以按照对应位置去填充给出的字符串
    
    
    print("网站名：{name}, 地址 {url}".format(name="菜鸟教程", url="www.runoob.com"))
     
    # 通过字典设置参数
    site = {"name": "菜鸟教程", "url": "www.runoob.com"}
    print("网站名：{name}, 地址 {url}".format(**site))
     
    # 通过列表索引设置参数
    my_list = ['菜鸟教程', 'www.runoob.com']
    print("网站名：{0[0]}, 地址 {0[1]}".format(my_list))  # "0" 是必须的

f-string格式化字符串字面量
    
    
    >>> name = 'Runoob'
    >>> f'Hello {name}'  # 替换变量
    'Hello Runoob'
    
    >>> f'{1+2}'         # 使用表达式
    '3'
    
    >>> w = {'name': 'Runoob', 'url': 'www.runoob.com'}
    >>> f'{w["name"]}: {w["url"]}'
    'Runoob: www.runoob.com'

可以直接算出f'语句'花括号里面的值直接填进去输出
    
    
    x = 123
    print(f"x+1=")
    会直接输出x+1=124
